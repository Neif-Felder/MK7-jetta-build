## Custom Manifold — Engineering Story

The most involved part of this build. Designed a custom intake manifold in Fusion 360 from scratch.

**V1:** Steel printed by overseas manufacturer. Identified flange flatness failures causing boost leaks under pressure — flanges were not flat enough to seal properly.

**V2:** Redesigned flange geometry. Coordinated surface grinding with fabrication partner to achieve 0.001" flatness tolerance. Currently running and holding boost.

This project covered the full engineering loop:

> Design → Manufacture → Failure Analysis → Redesign → Validation
